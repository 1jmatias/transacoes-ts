# Dashboard de Transações Financeiras
Projeto desenvolvido como entregável da Trilha de TypeScript

## O que o dashboard exibe:

- Total de transações cadastradas
- Valor total somado de todas as transações
- Forma de pagamento mais utilizada
- Dia da semana com mais vendas
- Contagem por forma de pagamento (Cartão de Crédito / Boleto)
- Contagem por status (Paga, Recusada, Aguardando, Estornada)
- Tabela completa com nome, e-mail, valor, pagamento e status de cada transação

### Como Rodar o Projeto
Pré-requisitos: Node.js e NPM instalados.
Clone o repositório
git clone https://github.com/1jmatias/transacoes-ts.git

- Entre na pasta
cd transacoes-ts

- Compile o TypeScript
tsc

- Abra o index.html
npx live-server
